# Reference Exports

Generated reference snapshots and author-format `sol1_...txt` files belong
here. Keep raw `.npz` snapshots and exported text files grouped by case and grid
resolution, for example:

- `aei70_krar_80/`
- `aei700_krartr_80/`
- `lowres_from_previous_worker/`

Do not treat low-resolution exports as paper-level reproduction evidence.
