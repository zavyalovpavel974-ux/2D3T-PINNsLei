# Lei et al. 2025 论文实验逻辑讨论笔记

本文档整理了从“表一参数是什么”开始，围绕论文 *2D 3T PINNs for solving 2-D 3-T heat conduction equations based on physics-informed neural networks* 的讨论内容。重点是理解表 1、表 2、表 3，以及 Example 1-6 的递进关系：每个算例改变了什么、带来了什么困难、作者采用了哪些技术。

## 1. 表 1：材料物理参数

表 1 给出论文实验中使用的两种材料 `CH` 和 `SiO2` 的物理参数。

| 材料 | rho | Gamma_e | Gamma_i | Gamma_r | A_e | A_i | A_r | beta | A_ei | A_er |
| --- | ---: | ---: | ---: | ---: | ---: | ---: | --- | ---: | ---: | ---: |
| CH | 1.1 | 45 | 70 | 0.007568 | 81 | 0.02 | `2.1e2 / rho^2` | 3.0 | 7000 | 79 |
| SiO2 | 2.5 | 40 | 40 | 0.007568 | 60 | 0.00017 | `9.0e2 / rho^1.5` | 2.4 | 4000 | 140 |

参数含义：

- `rho`：材料密度。
- `Gamma_e, Gamma_i, Gamma_r`：电子、离子、光子温度方程中的热容相关系数。
- `A_e, A_i, A_r`：电子、离子、光子热导率中的比例系数。
- `beta`：光子热导率中温度幂次相关参数。
- `A_ei`：电子-离子能量交换耦合强度。
- `A_er`：电子-光子能量交换耦合强度。

需要注意：表 1 中 CH 的 `A_ei=7000` 是材料参数表中的代表值，但论文后续算例会人为设置不同 `A_ei`，例如 `70`、`400`、`700`，用于逐步增加问题难度。

## 2. 表 2：Example 2-5 的参数设置

表 2 是后续真实物理算例的路线图，列出 Example 2-5 中几个关键物理量的具体形式。

| 算例 | omega_ei | A_ei | omega_er | K_e | K_i | K_r |
| --- | --- | ---: | --- | --- | --- | --- |
| Example 2 | `A_ei rho T_e^(-2/3)` | 70 | `A_er rho T_e^(-1/2)` | `A_e T_e^(5/2)` | `A_i T_i^(5/2)` | `A_r` |
| Example 3 | `A_ei rho T_e^(-2/3)` | 400 | `A_er rho T_e^(-1/2)` | `A_e T_e^(5/2)` | `A_i T_i^(5/2)` | `A_r` |
| Example 4 | `A_ei rho T_e^(-2/3)` | 400 | `A_er rho T_e^(-1/2)` | `A_e T_e^(5/2)` | `A_i T_i^(5/2)` | `A_r T_r` |
| Example 5 | `A_ei rho T_e^(-2/3)` | 700 | `A_er rho T_e^(-1/2)` | `A_e T_e^(5/2)` | `A_i T_i^(5/2)` | `A_r T_r` |

表 2 的递进关系非常清楚：

```text
Example 2: Aei=70,  Kr=Ar        较容易
Example 3: Aei=400, Kr=Ar        电子-离子耦合增强
Example 4: Aei=400, Kr=Ar*Tr     光子热导率非线性增强
Example 5: Aei=700, Kr=Ar*Tr     强耦合和非线性叠加
```

其中：

- `omega_ei` 是电子-离子能量交换系数。
- `omega_er` 是电子-光子能量交换系数。
- `K_e, K_i, K_r` 分别是电子、离子、光子热导率。
- `A_ei` 增大意味着电子和离子之间能量交换更强，方程更刚性。
- `K_r` 从 `A_r` 变成 `A_r T_r` 后，光子扩散项变成非线性项。

表 3 说明：在有精确解的基础问题中，2D 3T PINNs 已经能把三种温度场误差控制在 `10^-2` 量级。它更像是方法有效性的初步验证。

## 4. 论文实验的总体逻辑

论文实验不是孤立堆算例，而是从简单到复杂逐步加难度，并在每一步引入对应技术。

整体逻辑可以概括为：

```text
先证明 PINNs 在简单 2D-3T 问题上能工作
-> 再逐步增加耦合强度和非线性
-> 每遇到原始 PINNs 难以处理的困难，就加入作者提出的改进技术
-> 最后组合这些技术处理更难的 2D-3T 热传导问题
```

作者提出或强调的技术包括：

1. 输出正性约束。
2. 初值损失和光子 Dirichlet 边界损失加权。
3. 初始条件损失取对数。
4. 迁移学习。
5. Fourier feature embedding。

其中，输出正性约束是所有算例的底层要求，因为三温模型中包含：

```text
T_e^(-2/3), T_e^(-1/2), T_e^(5/2), T_i^(5/2)
```

如果网络输出温度为负或接近非法区域，训练容易出现 `NaN`。

## 5. Example 1：有精确解的基础验证

### 设置变化

Example 1 构造了一个有精确解的二维三温热传导问题，材料使用 `SiO2`。它的目标是验证 2D 3T PINNs 框架本身是否能学习 `T_e, T_i, T_r` 三个温度场。

### 主要困难

论文指出 Example 1 的困难主要来自损失项尺度不平衡：

- 方程残差损失比其他损失大几个数量级。
- 光子残差损失又比电子、离子残差大约两个数量级。

如果不做加权，优化器会优先处理大的损失项，小的约束会被忽略。

### 采用技术

Example 1 使用：

- 输出正性约束。
- 损失加权策略。
- 双精度 `float64`。

权重设置为：

```text
lambda_re = lambda_ri = 100
lambda_rr = 1
lambda_brd = 1
lambda_0e = lambda_0i = lambda_0r = 1
```

这相当于提高电子和离子残差权重，使其与光子残差更接近。

## 6. Example 2：正性保持与损失加权

### 设置变化

Example 2 开始进入真实物理参数。材料换成 `CH`，没有精确解，使用传统数值解作为参考解。

参数设置：

```text
Aei = 70
Kr = Ar
```

这是较简单的真实物理问题：电子-离子耦合较弱，光子热导率为常数。

### 主要困难

主要困难包括：

- 温度必须保持为正，否则训练会出现 `NaN`。
- 单纯正性约束虽然能让模型训练，但精度还不够。
- 初始条件和光子边界条件容易被 PDE 残差损失淹没。

### 采用技术

Example 2 使用：

- 输出正性约束。
- 双精度训练。
- 损失加权策略。

权重设置为：

```text
lambda_re = lambda_ri = lambda_rr = 1
lambda_brd = 1000
lambda_0e = lambda_0i = lambda_0r = 10
```

其中 `lambda_brd=1000` 很重要，因为上边界 `T_r = 3e-4 + 2t` 是热量进入区域的关键驱动条件。

### 结果意义

表 4 显示 Example 2 达到期望精度：

```text
Te L2 ≈ 1.446e-2
Ti L2 ≈ 7.902e-3
Tr L2 ≈ 1.588e-2
```

这说明正性约束加损失加权可以让 2D 3T PINNs 在较简单真实参数下工作。

## 7. Example 3：增大 Aei，引入迁移学习

### 设置变化

Example 3 相比 Example 2 的主要变化是：

```text
Aei: 70 -> 400
Kr: 仍为 Ar
```

也就是说，光子热导率暂时不增加非线性，但电子-离子耦合显著增强。

### 主要困难

`Aei` 出现在：

```text
omega_ei = Aei rho T_e^(-2/3)
```

因此 `Aei` 增大意味着电子和离子之间能量交换更快、更强，方程更刚性，直接训练更困难。

### 采用技术

Example 3 使用：

- 输出正性约束。
- 损失加权。
- 初始条件损失取对数。
- 光子 Dirichlet 边界条件损失取对数。
- 迁移学习。
- 第二阶段降低学习率。

权重为：

```text
lambda_re = lambda_ri = lambda_rr = 1
lambda_brd = 20
lambda_0e = lambda_0i = lambda_0r = 10
```

迁移学习路径为：

```text
先训练 Aei=70，Adam 初始学习率 1e-3
再用该网络作为初值训练 Aei=400，Adam 初始学习率 1e-4
```

### 为什么引入对数损失

初始温度为：

```text
T0 = 3e-4
```

这个值很小，而后期温度可能达到 `O(1)`。如果直接使用普通 MSE，初始小温度条件容易被忽略。取对数后，初始小量级误差会更明显。

### 消融结果

表 5 显示：

- 去掉对数损失：出现 `NaN`。
- 去掉迁移学习：误差接近 `1`。
- 第二阶段不降低学习率：误差接近 `1`。
- 完整 2D 3T PINNs：误差达到 `10^-2` 到 `10^-3`。

因此 Example 3 说明：当电子-离子耦合增强时，迁移学习、对数损失和学习率降低是关键。

## 8. Example 4：增强 Kr 非线性，引入 Fourier Feature

### 设置变化

Example 4 相比 Example 3 的主要变化是：

```text
Aei = 400 不变
Kr: Ar -> Ar Tr
```

即电子-离子耦合保持不变，但光子热导率从常数变为温度相关。

### 主要困难

当：

```text
Kr = Ar Tr
```

光子扩散项变为：

```text
div(Kr grad Tr) = div(Ar Tr grad Tr)
```

这会增强非线性，并带来更明显的多尺度特征。普通 PINNs 存在低频偏置，可能难以捕捉高频或局部变化。

### 采用技术

Example 4 使用：

- 输出正性约束。
- 损失加权。
- 对数初始/边界损失。
- Fourier feature embedding。

权重沿用 Example 3：

```text
lambda_re = lambda_ri = lambda_rr = 1
lambda_brd = 20
lambda_0e = lambda_0i = lambda_0r = 10
```

Fourier feature embedding 的作用是向网络输入加入多频率信息，让网络更容易学习非线性增强后的空间变化。

### 消融结果

表 8 显示：

- 去掉对数损失：仍能达到一定精度，但误差变大。
- 去掉 Fourier feature：误差明显变大。
- 完整 2D 3T PINNs：结果最好。

完整方法的 L2 误差大致为：

```text
Te ≈ 1.781e-2
Ti ≈ 2.192e-2
Tr ≈ 9.195e-3
```

因此 Example 4 说明：当 `Kr` 的非线性增强时，Fourier feature embedding 对精度提升很重要。

## 9. Example 5：强耦合和非线性叠加，五种技术全部使用

### 设置变化

Example 5 是正问题中最复杂、最关键的算例。

相比 Example 4：

```text
Aei: 400 -> 700
Kr: 仍为 Ar Tr
```

这意味着两个困难叠加：

- 电子-离子耦合更强。
- 光子热导率仍是非线性的。

### 主要困难

作者测试后认为，当前 2D 3T PINNs 可解范围大致为：

```text
Aei <= 700
Kr = Ar Tr
```

如果进一步到：

```text
Kr = Ar Tr^(3+beta)
```

或：

```text
Aei > 700
```

当前方法就难以达到合理精度。作者观察到多个损失项无法同时降到 `1e-2` 以下，损失之间会发生冲突。

### 采用技术

Example 5 使用作者提出的五种技术：

1. 输出正性约束。
2. 损失加权。
3. 对数初始/边界损失。
4. 迁移学习。
5. Fourier feature embedding。

权重沿用 Example 3：

```text
lambda_re = lambda_ri = lambda_rr = 1
lambda_brd = 20
lambda_0e = lambda_0i = lambda_0r = 10
```

迁移学习路径为：

```text
Aei=70，Adam 初始学习率 1e-3
-> Aei=400，Adam 初始学习率 1e-4
-> Aei=700，Adam 初始学习率 1e-4
```

### 消融结果

表 10 显示：

- 去掉迁移学习：误差明显变大。
- 去掉 Fourier feature：误差接近 `1`，基本失败。
- 完整 2D 3T PINNs：达到 `10^-2` 到 `10^-3`。

表 9 中完整方法 L2 误差大致为：

```text
Te ≈ 1.485e-2
Ti ≈ 1.738e-2
Tr ≈ 4.216e-3
```

因此 Example 5 的结论是：面对强耦合、强非线性和多尺度问题，必须组合使用五种技术。

## 10. Example 6：反问题，反演密度 rho

### 设置变化

Example 6 不再是单纯正问题，而是数据驱动的 PDE 参数发现问题。

它基于 Example 2：

```text
Aei = 70
Kr = Ar
```

但不同之处是：

```text
rho 未知
rho 作为可训练参数
```

作者加入五个时刻的参考解数据作为监督：

```text
t = 1e-5, 0.3, 0.5, 0.7, 1
```

每个时刻取 100 个监督点。

### 主要困难

反问题中，网络不仅要拟合 `T_e, T_i, T_r`，还要通过 PDE 残差反推出未知物理参数 `rho`。如果监督点不足或损失权重不合适，参数和解场可能同时偏移。

### 采用技术

Example 6 使用：

- 输出正性约束。
- Example 2 的损失权重设置。
- 五个时刻的监督数据损失。
- 将 `rho` 设为可训练参数。

监督损失权重为：

```text
supervised loss weight = 1000
```

其他损失权重与 Example 2 相同。

### 结果意义

表 12 显示真实值为：

```text
rho = 1.1
```

不同初始值下预测为：

```text
初始 rho = 0.5 -> 预测 rho = 1.11737
初始 rho = 1.0 -> 预测 rho = 1.11717
```

相对误差约为：

```text
1.579%
1.561%
```

这说明 2D 3T PINNs 不仅可以求解正问题，也可以在少量监督数据帮助下反演 PDE 参数。

## 11. 总结表

| Example | 主要变化 | 主要困难 | 采用技术 |
| --- | --- | --- | --- |
| Example 1 | 构造精确解，`SiO2` | 残差损失尺度不平衡 | 正性约束、损失加权、`float64` |
| Example 2 | `CH`, `Aei=70`, `Kr=Ar` | 温度正性、初边界损失弱 | 正性约束、损失加权 |
| Example 3 | `Aei=70 -> 400` | 耦合增强、方程更刚性 | 对数损失、迁移学习、降低学习率 |
| Example 4 | `Kr=Ar -> ArTr` | 光子扩散非线性增强 | Fourier feature、对数损失 |
| Example 5 | `Aei=700`, `Kr=ArTr` | 强耦合、非线性、多尺度叠加 | 五种技术全部使用 |
| Example 6 | `rho` 未知 | 正问题和参数反演耦合 | 监督数据损失、`rho` 可训练 |

一句话概括：

```text
Example 2 解决“能不能稳定训练”；
Example 3 解决“耦合增强怎么办”；
Example 4 解决“非线性增强怎么办”；
Example 5 解决“耦合和非线性同时增强怎么办”；
Example 6 证明“该框架还能做反问题”。
```
